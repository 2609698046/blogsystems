# blogsystems
个人博客系统 响应式布局
SpringBoot2.1.7版 个人博客系统
主要分为前后台，前台浏览博客，浏览电影推荐，给博主留言(只有审核通过的留言才会显示在前台页面上)
后台用到的是H-UI admin框架.上传图片,WangEditor富文本发布博客,发布电影,PageHelper分页插件,验证码,审核留言,编辑博主资料,增删改查.等好多功能自己运行查看即可.运行时部分截图如下:
